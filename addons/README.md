# Add-ons

Optional **dev tools** for maintaining the game. They are **not linked from the party UI** — guests only see `index.html`.

| Add-on | What it does |
|--------|----------------|
| [hint-tuner](./hint-tuner/) | Pick hint start times on each full MP3 using a waveform; saves to `playlist.json` |

Add-ons read and write the same repo root as the game (`playlist.json`, `clips/`, `scripts/`). See each add-on’s README for setup and usage.
